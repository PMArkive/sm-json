---
Title: Rename
---

Renames the key of an existing item in the object.

#- Arguments
- from `const char[]`
- Existing key to rename.
- to `const char[]`
- New key.
- replace `bool`
- Should the 'to' key should be replaced if it exists? *default:* `true`

#- Returns
- `bool`
- True on success, false otherwise.
